## Steps to Add Screenshot

### Method 1: Using Screenshot Tool (Mac)
1. Open your application at http://localhost:3001
2. Navigate to Admin Panel (login as admin)
3. Take a screenshot:
   - Press `Cmd + Shift + 4` to capture a selection
   - Or press `Cmd + Shift + 3` for full screen
   - Or press `Cmd + Shift + 4` then `Space` to capture a window
4. The screenshot will be saved to your Desktop
5. Drag and drop it to: `INCUBYTE/screenshots/admin-panel.png`
   - Or copy from Desktop and paste into the screenshots folder

### Method 2: Using Finder
1. Take screenshot of your Admin Panel
2. Open Finder
3. Navigate to: `Desktop/INCUBYTE/screenshots/`
4. Drag your screenshot file there
5. Rename it to: `admin-panel.png`

### Method 3: Using VS Code/Cursor
1. Take screenshot and save it somewhere
2. In VS Code/Cursor, right-click on `screenshots/admin-panel.png`
3. Choose "Replace" or "Save As"
4. Select your screenshot image

### Method 4: Using Terminal
```bash
# If you have the screenshot saved somewhere, copy it:
cp ~/Desktop/your-screenshot.png screenshots/admin-panel.png

# Or move it:
mv ~/Desktop/your-screenshot.png screenshots/admin-panel.png
```
